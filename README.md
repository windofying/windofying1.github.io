# CECC Website

Constant Energy Construction Corp (CECC) 官方网站

## 项目描述

CECC 是一家提供工业电气、自动化、可再生能源、数据中心电源和通信解决方案的全方位服务电气和基础设施建筑承包商。

## 技术栈

- HTML5
- CSS3
- JavaScript
- 响应式设计

## 页面结构

- `index.html` - 首页
- `about.html` - 关于我们
- `services.html` - 解决方案与服务
- `projects.html` - 项目展示
- `safety.html` - 安全与资质
- `contact.html` - 联系我们
